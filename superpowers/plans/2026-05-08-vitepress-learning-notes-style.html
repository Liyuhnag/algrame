VitePress Learning Notes Style Implementation Plan ​

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

Goal: Convert the starter VitePress site into a polished Chinese learning-notes knowledge base for algorithm solutions and Java backend interview answers.

Architecture: Keep the VitePress default theme and extend it with a local theme entry, one lightweight Vue slot component for the homepage 笔记速览 panel, and a focused stylesheet. Replace starter navigation and sample copy with three durable content entry pages: algorithms, interview, and topics.

Tech Stack: VitePress 1.6.4, Vue 3 theme slot API, Markdown, CSS variables.

File Structure ​

  • Modify docs/.vitepress/config.mts: site title, description, Chinese nav, grouped sidebar, local search, and outline labels.
  • Create docs/.vitepress/theme/index.ts: extend the default theme and fill the home hero image slot with NoteOverview.
  • Create docs/.vitepress/theme/components/NoteOverview.vue: homepage right-side 笔记速览 panel.
  • Create docs/.vitepress/theme/custom.css: visual system for navbar, sidebar, homepage, markdown content, code blocks, and responsive behavior.
  • Modify docs/index.md: homepage copy, actions, and feature cards.
  • Create docs/algorithms/index.md: algorithm section landing page with usable headings.
  • Create docs/interview/index.md: interview section landing page grouped by technical direction.
  • Create docs/topics/index.md: topic and review section landing page.
