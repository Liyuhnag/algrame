# 专题索引

专题页用于把算法题解和面试题库中反复出现的知识点串起来，形成阶段复习路线。

## 项目追问

项目追问记录自己项目里最容易被问深的部分。每个专题建议包含：

- 背景：为什么项目需要这个设计
- 方案：当时如何实现，有哪些取舍
- 追问：面试官可能继续问什么
- 复盘：有没有更好的实现方式

## 场景设计

场景设计用于沉淀缓存一致性、接口幂等、限流降级、消息可靠性、分库分表等问题。

建议每篇场景题按下面顺序组织：

1. 先澄清业务目标和约束
2. 再画出核心流程或数据流
3. 说明关键风险和兜底策略
4. 最后给出可以落地的简化方案
